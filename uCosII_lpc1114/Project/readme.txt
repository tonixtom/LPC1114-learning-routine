Chip:
    LPC1114F301

Evaluation Board:
    RuiCST-LPC11XX

Development Environment:
    Keil RealView MDK 4.12

Complier:
    ARMCC

Tasks:

    The project is a simple program of LED Blink for the LPC11XX based on ucOS.

    XTAL   freq        =  12.00 MHz
    SYSCLK freq       =  48.00 MHz
    System Tick freq = 100Hz (10ms)
